# Fomo-Widget Landing Page

A single-page marketing site for **Fomo-Widget**—social proof and FOMO notifications that integrate into your website to increase stickiness, trust, and conversions.

## Features

- **SEO-friendly**: Semantic HTML, meta tags, Open Graph, Twitter cards, and JSON-LD schema
- **Modern layout**: Dark theme, gradient orbs, grid pattern, responsive design
- **Sections**: Hero, features (social proof & FOMO), how it works, testimonials, FAQ, CTA
- **Accessible**: ARIA labels, keyboard-friendly FAQ accordion, focus states

## Run locally

No build step required. Open `index.html` in a browser or use a local server:

```bash
# Python 3
python3 -m http.server 8080

# Node (npx)
npx serve .
```

Then visit `http://localhost:8080`.

## Assets

## Deploy

Upload the repo (or just `index.html`, `styles.css`, `main.js`) to any static host (Netlify, Vercel, GitHub Pages, etc.). Set the canonical URL and OG URL to your production domain in `index.html`.
